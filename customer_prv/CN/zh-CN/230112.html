1.简介

 

LG Electronics Inc. 及其公司集团（统称为 “LGE” 或“我们”）致力于保护使用 Sync On Mobile 应用程序的个人或团队的信息。本隐私政策规定我们如何收集、使用和共享因您使用这些服务而获得的信息，尤其规定了：

 

• 因您使用 Sync On Mobile 应用程序，LGE 可从您那里获得的信息内容；

 

• LGE 如何使用获得的信息；

 

• LGE 何时将这些信息分享给他人；

 

• 您如何控制 LGE 对您个人信息的使用和共享。

 

2.LGE 获得了什么信息？

 

要使用 Sync On Mobile 应用程序，您的移动设备和个人电脑必需连接至同一个 Wi-Fi 网络。要获得 Sync On Mobile 应用程序的全部内容，我们需要使用您的个人信息，以促进在 Sync On Mobile 应用程序和我们系统之间实现更多的信息交换。如果您不希望将您的信息用于这种方式，那么您将不能使用 Sync On Mobile 应用程序的全部内容。下文将描述我们如何使用各类别信息（以及该信息可能会在何时识别您的身份）的相关内容。

 

在您使用 Sync On Mobile 应用程序时我们可能会收到的信息分为以下几类：

 

• LGE 可能会收到您为了访问 Sync On Mobile 应用程序而同意提供的信息。这类信息可能包括推送 Token（令牌，接收 Sync On Mobile 应用程序的推送通知时收集的信息）和日志信息（访问和使用日志）

 

• 自动化数据收集。  我们、我们的服务供应商以及业务伙伴可能会自动记录关于您、您的电脑或移动设备以及您和本服务、我们的通讯及其他在线服务的长期互动等信息，如：

•

Google 分析。我们可能会长期通过第三方网站、设备、应用程序和其他在线功能和服务，收集与您在网站和连接设备上的在线活动相关的个人信息。我们可能使用 Google 分析和 Firebase 等第三方分析服务。我们获得的信息可能会向服务商和其他相关第三方披露，或直接从他们处收集，供他们用来评估服务的使用以及帮助管理 Sync On Mobile 应用程序。如需了解更多关于 Google 分析的信息，请访问“在您使用我们合作伙伴的网站或应用程序时，Google 如何使用数据”（网址为 https://www.google.com/policies/privacy/partners/)，“保护您的数据”（网址为 https://support.google.com/analytics/answer/6004245#zippy=) 以及“Google 分析选择退出浏览器插件”（网址为 https://tools.google.com/dlpage/gaoptout）。

 

3.LGE 如何使用获得的信息？

 

我们从 Sync On Mobile 应用程序获得的信息（非个人信息和个人信息，如适用）将用于多个目的，包括：

 

• 提供 Sync On Mobile 应用程序；

 

• 分析、审查和改进我们提供的产品和服务；

 

• 了解顾客如何使用 Sync On Mobile 应用程序；

 

• 开发新产品和服务

 

• 为您提供用户友好型导览体验；

 

• 保护并捍卫我们的权利和财产，包括通过法律诉讼；

 

• 用于内部记录保存

 

根据您所在国家/地区和您拥有的 Sync On Mobile 应用程序版本，有部分功能可能无法使用，还有部分功能可能拥有不同的名字，并可能与其他功能相结合。

 

LGE 保留从 Sync On Mobile 应用程序添加或删除服务，以及不时修改 Sync On Mobile 应用程序的权利。

 

我们如何使用您的会员信息

 

您的会员信息可能用于向您提供需要 Sync On Mobile 应用程序会员的 Sync On Mobile 应用程序。

 

4.信息共享

 

我们一般不与第三方共享因您使用 Sync On Mobile 应用程序而收集到信息。但是，请注意在以下情况下我们可能会与第三方共享此类信息：

 

• LGE 只会在必要时，与全球各地的联属和附属公司共享个人信息和非个人信息（包括会员信息），并且只用于信息收集的目的

 

• 根据法院命令或其他政府或执法机构的要求，以遵守法律程序或法律；

 

• 与企业活动相关的情况下，如并购或 LGE 整体或部分业务出售（包括部门或附属公司的业务），此时，LGE 可能需要向潜在买方和其专业顾问披露信息；

 

• 为了保护并捍卫我们的权利和财产，包括通过法律诉讼；以及

 

• 为了保护我们用户或公众的个人安全或财产安全。

 

LGE 将采取合理的商业措施，以确保任何此类第三方稳妥地处理您的个人信息。

 

5.跨境传输

 

作为一个全球组织，我们、我们的联属和附属公司以及供应商可能需要将您的个人信息（包括您的个人信息）传输至 Sync On Mobile 应用程序所在国家以外的地方，如我们在韩国的总部。我们可能在遵守本隐私政策和任何适用法律的情况下，传输您的个人信息以便向您提供我们的 Sync On Mobile 应用程序，或本隐私政策概述的任何其他目的。您还应记住在这些国家中，许多国家的数据保护法可能无法提供与您祖国相同的保护水平。但是，传输前我们将采取措施来确保已采取保护措施，从而确保保护水平与 Sync On Mobile 应用程序所在国家的适用数据保护法规定的相同。同意本隐私政策，即表示您明确同意我们、我们的业务伙伴及供应商在任何司法管辖区内（包括但不限于韩国和美国），根据本隐私政策处理您的数据。如需了解关于我们如何处理和保护您个人信息的更多信息，请通过电子邮件 [lg-synconmoible@lge.com] 提交您的疑问或意见。

6.LGE 会将个人信息保存多长时间？

 

我们将采取合理措施确保我们保存您的个人信息不超过我们为您提供 Sync On Mobile 应用程序、或其收集目的或适用法律所要求的必要期限。

 

我们采用标准来确定您提供给我们的各类别个人信息的适当保留期限。例如，我们将保留能在法律索赔中保护我们的必要信息，如与合同有关的信息，保留期限为合同有效期及之后七年内。此后，我们将删除这些信息或在某些情况下进行匿名化处理。我们在您同意的基础上收集个人信息，且没有其他合法依据来继续处理，如果您撤回同意，我们将删除您的个人信息。

 

7.Cookie 同意

 

Cookie 是一个小型的数据文件，通常包含一个存储在 Sync On Mobile 应用程序上的唯一设备识别符。我们可能使用 Cookie 来检查服务的连接状态。

 

8.安全

 

我们采用合理的商业努力来保护您个人信息的机密性和安全性。然而，由于互联网的设计、不断变化的技术和其他超出我们控制的因素，我们不能保证您和我们服务器之间的通信不会遭受第三方未经授权的访问，也不能保证我们不会受到安全漏洞的影响。对于因传输错误或第三方的未授权或不法行为而造成的个人信息泄露，我们不承担任何责任。

 

9.保护儿童隐私

 

保护儿童隐私对我们来说非常重要。因此，我们 Sync On Mobile 应用程序皆不针对或旨在吸引未满 16 岁的儿童。我们也不会有意收集或保存从未满 16 岁的用户处收集到的个人信息。一旦我们了解或被告知我们收集了未满 16 岁用户的信息，我们会及时删除这些个人信息。

 

10.代表他人同意

 

如果您其他家庭成员或访客使用了 Sync On Mobile 应用程序，而且您同意本隐私政策，则您已获得他们对本隐私政策的同意并代表他们提供同意。如果 Sync On Mobile 应用程序的用户未满 18 岁，且您同意本隐私政策，则您代表他们同意本隐私政策。

 

11.LGE 隐私政策的更改

 

LGE 可能会不时更新本隐私政策。如果我们对我们的隐私政策做出重大更改，我们将通过 Sync On Mobile 应用程序或其他方式向您提供更新。我们可能要求您确认或同意修订后的隐私政策，以便继续使用某些 Sync On Mobile 应用程序。

 

12.就本隐私政策联系 LGE

 

如果您对本隐私政策或 LGE 处理您的个人信息有任何疑问或意见，请通过电子邮件 [lg-synconmobile@lge.com] 提交您的疑问或意见。您有权向您居住国的数据保护机构提出投诉，但我们鼓励您先直接联系我们以解决任何问题。

 

13.个人信息的访问和控制

 

除法律另行规定外，对于我们保管和控制的您的个人信息，您有权要求我们删除、访问、更新和纠正其中的不准确之处，您还有权获得您个人信息的副本。您还有权反对我们使用您个人信息（包括用于直接营销目的）的方式，并撤回您授予我们的任何同意。您可随时通过上述联系信息向我们发送电子邮件或书信来行使这些权力。对于寻求访问其个人信息记录的个人，我们可能要求提供某些能够核实其身份的个人信息。

 

14.转让

 

在我们部分或全部资产被另一方出售或收购，或在并购的情况下，您授予我们转让权，来转让因您使用 Sync On Mobile 应用程序而收集到的信息。
